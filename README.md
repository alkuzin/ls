# ls

Simple implementation of ls command for Unix-like operating systems

<img src="res/ls.png">

## Installation

Clone this repository:
```console
git clone https://github.com/alkuzin/ls.git
```

Build the project:
```console
mv ls && make all
```

To get list of all available flags:
```console
./ls --help
```
